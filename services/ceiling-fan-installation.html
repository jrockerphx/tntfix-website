Ceiling Fan Installation or Replacement

$179 (swap) / $299 (new install) | 1.5 hours (swap) / 2.5 hours (new install)

Home Repair Electrical $179 – $299 Serving Phoenix Metro, Coolidge & Casa Grande

Overview

  • Duration: 1.5 hours (simple swap) / 2.5 hours (new install)
  • Price: $179 (swap) / $299 (new install)
  • Description: Beat the Arizona heat efficiently! We'll install your new ceiling fan properly and safely. Perfect for bedrooms, living rooms, and patios.

Simple Swap Service ($179) Includes

  • Remove existing ceiling fan
  • Install new fan (customer provides)
  • Connect to existing wiring and junction box
  • Balance and test all functions
  • Program remote control if included
  • Haul away old fan
  • Best for: Replacing an old/broken fan with a new one (existing electrical box rated for fans)

New Installation Service ($299) Includes

Everything in Simple Swap PLUS:

  • Install fan-rated ceiling box if needed
  • Run new wiring from existing switch (if accessible)
  • Patch and paint ceiling around box (customer provides paint)
  • Best for: Adding a fan where none existed, or ensuring safe installation with the correct box

Popular Upgrades (+cost)

  • Smart fan control installation: +$120 (Bond, Hunter WiFi module, etc.)
  • Separate wall controls: +$80 (if accessible wiring)
  • Light kit installation: +$40

Not Included

  • Fans over 60" diameter (+$50 for large fans)
  • Cathedral/vaulted ceiling mounts (+$80 for extension rod + complexity)
  • Running a new circuit from the breaker (requires consultation)
  • Outdoor fan installation in a covered area: same price

Safety Note

We'll verify your ceiling box is fan-rated. If it's not, we’ll install a proper fan-rated box (included in the New Install price).
